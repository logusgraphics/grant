# Feature Comparison

#### Core RBAC/ACL

- ✅ Basic user management
- ✅ Role-based access control
- ✅ Organization and project scoping
- ✅ Permission management
- ✅ Group management
- ✅ Basic audit logging

#### Self-Hosting

- ✅ CloudFormation templates
- ✅ Docker containers
- ✅ AWS deployment
- ✅ Basic monitoring
- ✅ Community support

#### API & SDK

- ✅ Core API endpoints
- ✅ TypeScript SDK
- ✅ Basic middleware
- ✅ GraphQL API

## Migration Path

### From Open Source to SaaS

1. **Start with Core** - Use open source for basic needs
2. **Scale Up** - Migrate to SaaS for advanced features
3. **Data Migration** - Seamless migration tools provided
4. **Support** - Dedicated migration support

### Pricing Strategy

- **Open Source**: Free forever
- **SaaS Starter**: $99/month (up to 100 users)
- **SaaS Professional**: $299/month (up to 1000 users)
- **SaaS Enterprise**: Custom pricing (unlimited users)
